Trajetória

Cada etapa moldou um olhar analítico único — da sala de aula ao cliente global.

2021 – 2023
O começo: fundação técnica sólida
Senac · Técnico em Informática
Tudo começou com uma curiosidade genuína por tecnologia. No Senac, aprendi os fundamentos que ninguém te ensina depois: SQL desde o zero, administração de servidores Windows, Linux e a lógica por trás da automação com Python. Mais do que técnica, foi onde desenvolvi o pensamento sistemático de quem precisa resolver problemas reais com ferramentas reais.
SQL Python Linux Office 365 Windows Server
2023 – Atual
Ciência da Computação: dados em escala
SPTECH · Foco em Machine Learning & Reconhecimento de Padrões
Na graduação, fui além da teoria. Em projetos reais com AWS, ETL/ELT, Grafana e análise preditiva, aprendi que dado sem contexto é ruído — e dado com contexto é poder. O projeto Elder Care monitorou saúde de idosos em tempo real. O Rise conectou ONGs a pessoas em situação de rua via geolocalização. O TechIdeas criou dashboards de KPIs correlacionados em múltiplas plataformas. Cada projeto foi um laboratório de dados aplicado a impacto social.
AWS ETL/ELT React Jupyter Grafana Python R
Mar – Out 2024
Primeiro passo na Accenture: dados e inovação
Accenture · IT Innovation Intern
Minha entrada no mercado corporativo foi direto no coração da inovação. Desenvolvi uma calculadora de carbono rodando em AWS com Linux Server, construí dashboards em Power BI para geração de insights de sustentabilidade e entreguei análises acionáveis para stakeholders. Foi aqui que entendi a diferença entre visualizar dados e contar histórias com dados.
Power BI AWS EC2 JavaScript Linux
Out 2024 – Jun 2025
Cliente global: risco, compliance e dados em produção
Accenture · SAP Security Intern
Ainda como estagiário, o salto para projetos internacionais foi intenso — e exatamente o que eu almejava. Atuei em dois projetos paralelos da Sanofi (farmacêutica global), comandando reuniões em inglês e espanhol, elaborando requisitos funcionais, análises preventivas de risco de acesso (SoD) e suporte em SIT, UAT e Go-Live com alto volume de incidentes. Aprendi que dados de acesso errado custam caro — e que análise preventiva é o diferencial entre caos e controle.
SAP Security SoD Analysis UAT/SIT Compliance SQL
Jun 2025 – Atual
Analista Júnior: resultados mensuráveis
Accenture · SAP Security Junior Analyst
Promovido para Analista Júnior, passei a liderar análises de dados de risco (SoD) com SQL e Excel, identificando padrões que reduziram em ~30% as inconsistências durante UAT e Cutover. Otimizei o fluxo de requisições de acesso (ARM), estruturei documentação de processos e transformei dados complexos em insights claros para stakeholders. Cada análise tem um impacto direto no negócio — e é assim que gosto de trabalhar.
SQL Avançado Data Analysis Risk Management Documentação Stakeholders
